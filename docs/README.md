# 📚 Documentación del Proyecto - MussikOn

## 🎯 **Descripción General**

Esta carpeta contiene toda la documentación técnica del proyecto MussikOn, una aplicación móvil desarrollada en React Native con Expo para conectar músicos y organizadores de eventos musicales.
